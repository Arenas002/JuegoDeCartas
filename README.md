# Marvel Game

## Objetivos del proyecto

Este proyecto se baso en el patron de DDD(Domain Driven Desing) un patron que se orienta en el modelamiento de dominio, donde se hace un analisis sobre la necesidad puntual del negocio

* Analisis del modelamiento de dominio centrandose en el problema 

* Realizar casos de usos y test por comportamiento

* Programacion funcional y reactiva.
